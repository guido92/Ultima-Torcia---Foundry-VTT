# L'Ultima Torcia - Foundry VTT System

Un sistema per Foundry VTT dedicato al gioco di ruolo "L'Ultima Torcia".

## Funzionalità

*   **Scheda Personaggio**: Gestione completa di attributi, risorse (Torce, Razioni, ecc.), equipaggiamento e note.
*   **Tiri di Dado**: Supporto per tiri normali, vantaggio (2d10kh1) e svantaggio (2d10kl1) direttamente dalla scheda.
*   **Oggetti**: Schede dedicate per Armi, Armature, Consumabili, Abilità e Incantesimi.
*   **Localizzazione**: Completamente in Italiano.

## Installazione

1.  Copia l'URL del `system.json` (Manifest URL).
2.  In Foundry VTT, vai su "Game Systems".
3.  Clicca "Install System".
4.  Incolla l'URL nel campo "Manifest URL".
5.  Clicca "Install".

## Sviluppo

Per contribuire o modificare il sistema:

1.  Clona questo repository nella cartella `Data/systems/` della tua installazione di Foundry VTT.
2.  Avvia Foundry VTT.
